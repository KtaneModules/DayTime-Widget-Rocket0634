While manuals are made in HTML, some people like to be able to print out your manual. Make sure to use Google Chrome's Print-to-PDF feature to convert this manual to a PDF. Make sure to release both the HTML and PDF versions of your manual.

You can do this as follows:

  1. Right-click and select "Print..." or press Ctrl+P
  2. Change the destination to "Save as PDF"
  3. Make sure the "Background graphics" option is checked so that the PDF will include background images such as the fake smudges.
  4. Then press "Save" to save the PDF.
